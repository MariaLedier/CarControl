# 🚛 Sistema de Gestão de Frotas — Estágio

Sistema web desenvolvido para gerenciamento de frotas de veículos, incluindo controle de abastecimento, manutenção, pneus e checklists.

## 🛠️ Tecnologias Utilizadas

**Backend**
- Node.js + Express
- MySQL
- JWT para autenticação

**Frontend**
- Next.js 15
- Bootstrap 5

**Infraestrutura**
- Oracle Cloud Infrastructure (OCI)
- PM2 + Nginx

## 📋 Funcionalidades

- Cadastro e gerenciamento de veículos
- Controle de abastecimento
- Registro de manutenções
- Controle de pneus e rodízio
- Checklist de veículos
- Relatórios de abastecimento, manutenção e pneus
- Autenticação com JWT
